1. Descargar de github https://github.com/DannyVegaB/devsu_demoblaze.git
2. Abrir en visual studio code el proyecto devsu
3. Ejecutar los comandos 
	- npm init
	- npm install cypress --save-dev
	- npm i -D cypress prettier
	- npm install --save-dev typescript
	- npm install @badeball/cypress-cucumber-preprocessor
	- npm install @bahmutov/cypress-esbuild-preprocessor
4. finalmente escribir el comando npx cypress open
